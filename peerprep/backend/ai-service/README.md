# AI Service
A microservice for generative ai features in Peerprep.

## Overview

The AI service uses Google's Gemini API to enable Peerprep users to receive AI assistance.

## Features

- Receive prompts from users and generate a responses
- Take in past attempt code and generate feedback

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

```bash
npm install

```

### Environment Variables

Create a `.env` file by referencing the `.env.example` file and set the necessary environment variables.

### Running the Service

```bash
# Development
npm run dev

# Production
npm start
```
